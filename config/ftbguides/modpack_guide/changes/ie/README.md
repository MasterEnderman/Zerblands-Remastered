# Immersive Engineering

## Wires

Increased the percentage of **power lost** every 16 blocks of distance for the wire tiers.

Copper: 70% loss rate
Electrum : 50% loss rate
Steel: 30% loss rate

Decreased the **transfer rates** in Flux/t for the wire tiers.

Copper: 64 Flux/t
Electrum : 256 Flux/t
Steel: 1024 Flux/t
